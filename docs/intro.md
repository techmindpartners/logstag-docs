# Introduction

Welcome to the **Logstag** documentation!

Logstag is an AI-powered database monitoring platform that provides actionable insights, powerful analytics, and proactive alerts for multiple database engines.

## Why Logstag?

Modern applications rely on complex data infrastructures. Logstag simplifies monitoring by offering:

- **AI-Powered Insights**: Understand *why* performance issues happen, not just *what* happened.
- **Unified Observability**: Monitor PostgreSQL, Microsoft SQL Server, Oracle, MongoDB, and more from a single dashboard.
- **Smart Alerts**: Get prioritized notifications based on risk and context, avoiding alert fatigue.
